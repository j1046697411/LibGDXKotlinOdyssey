## 一、主界面设计

### 1.1 弟子信息面板

**设计说明**：主界面左上角显示当前角色的核心信息，简洁明了，便于玩家随时查看。

**界面示例**：
```
┌───────────────────────────────────────┐
│ 【林玄风】- 青云宗内门弟子            │
├───────────────────────────────────────┤
│ 修为：金丹中期 | 气血：2850/2850 💗    │
│ 灵力：1240/1800 💠 | 寿元：128年 ⏳    │
└───────────────────────────────────────┘
```

### 1.2 宗门管理面板

**设计说明**：主界面右侧显示宗门管理相关选项，便于玩家快速访问宗门功能。

**界面示例**：
```
┌─────────────────┐
│   宗门管理      │
├─────────────────┤
│ ▶ 宗门总览      │
│ ▶ 弟子管理      │
│ ▶ 资源管理      │
│ ▶ 设施建设      │
│ ▶ 任务大厅      │
│ ▶ 功法堂        │
│ ▶ 炼丹房        │
│ ▶ 炼器房        │
└─────────────────┘
```

### 1.3 任务列表面板

**设计说明**：主界面下方显示当前可接任务和已接任务，便于玩家管理任务进度。

**界面示例**：
```
┌───────────────────────────────────────┐
```
