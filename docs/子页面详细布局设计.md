# 子页面详细布局设计

## 一、设计原则

1. **统一风格**：所有子页面使用相同的ASCII边框和布局结构
2. **信息分层**：核心信息突出显示，次要信息有序排列
3. **导航一致**：每个页面包含返回和主页按钮
4. **功能完整**：包含该页面所需的所有核心功能
5. **视觉引导**：通过图标和颜色增强视觉效果

## 十一、设计说明

1. **统一风格**：所有页面使用相同的ASCII边框字符（┌┐└┘├┤┼）
2. **信息层次**：核心信息位于左上角和中间位置，次要信息位于右侧
3. **导航一致**：每个页面顶部包含导航路径，底部包含返回和主页按钮
4. **功能完整**：每个页面包含该功能所需的所有核心操作
5. **视觉增强**：使用emoji图标和颜色增强视觉效果
6. **响应式设计**：布局可根据屏幕大小调整，保持信息完整

## 十二、应用建议

1. **开发实现**：使用ECS组件系统管理页面状态
2. **动态更新**：实时更新进度条和状态信息
3. **交互反馈**：按钮点击有明显的视觉反馈
4. **个性化设置**：允许玩家调整信息显示密度
5. **无障碍设计**：支持键盘导航和屏幕阅读器

通过这套详细的子页面布局设计，玩家能够清晰地理解每个页面的功能和操作方式，减少学习成本，提升游戏体验。设计保持了统一的风格和清晰的信息层次，便于开发团队实现和扩展。
