# 整体布局样式设计

## 一、设计原则

1. **1920×1080优化**：针对160字符×60行分辨率优化布局
2. **视觉焦点比例**：左:中:右 = 2:5:2
3. **清晰层次**：通过边框、分隔线和缩进区分不同功能区域
4. **信息模块化**：将相关信息分组展示，便于用户快速获取
5. **交互友好**：提供明确的操作选项和状态反馈
6. **视觉平衡**：合理分配界面空间，避免信息拥挤
7. **快捷键支持**：提供常用操作的快捷键，提升操作效率
8. **状态可视化**：使用图标和颜色（通过文本符号模拟）表示不同状态
9. **响应式设计**：动态布局调整、可折叠区域设计，适应不同屏幕尺寸
10. **聚焦核心**：简化界面，聚焦核心功能，提升用户体验
11. **信息密度优化**：合并重复区域、精简顶部导航栏、折叠次要信息，减少视觉疲劳
12. **导航效率提升**：优化菜单层级、增加快速搜索、增强快捷键系统，减少操作步骤
13. **视觉层次强化**：统一图标系统、优化区域间距、颜色编码，提升可读性和美观度

## 三、区域拆分与功能说明

### 1. 顶部导航栏

**位置**：界面顶部
**功能**：显示当前区域、天气、游戏时间和游戏速度等核心信息，提供快速搜索和详情展开
**设计要点**：
- 紧凑模式：合并显示区域、天气、时间和游戏速度
- 详情模式：点击展开显示完整信息（季节、详细时间、区域效果）
- 集成搜索框，支持按/键快速聚焦
- 信息清晰，便于快速获取游戏状态
- 使用图标增强视觉效果
- 支持游戏速度调节


**位置**：界面左侧

**设计要点**：

- 使用数字快捷键（1-9,0）快速访问
- 简洁的图标和文字结合
- 图标直观反映功能类型
- 当前选中项高亮显示
- 功能分类清晰，便于快速定位
- 支持键盘快捷键快速切换

## 五、设计应用建议

### 1. 不同游戏模式适配

- **掌门模式**：左侧菜单增加「宗门决策」、「外交关系」等选项
- **长老模式**：左侧菜单聚焦于管理的专项领域
- **弟子模式**：左侧菜单简化，增加「个人修炼」、「任务跟踪」等选项
- **观察者模式**：左侧菜单增加「数据分析」、「世界观察」等选项

- **大屏幕**：完整显示所有区域
- **中屏幕**：可折叠右侧信息区
- **小屏幕**：左侧菜单可滑出隐藏

- 使用颜色区分不同功能区域
- 重要信息使用闪烁或高亮效果
- 交互元素添加悬停反馈
- 使用动画过渡效果增强体验

- 使用ECS组件系统管理各区域状态

- 实现模块化设计，便于后期扩展
